# Day 2 教学文档：第一次真正跑 FashionMNIST

这一天最重要的目标不是追求最高准确率，而是把“真实图片数据集 + 完整训练流程”真正看懂。

## 1. 这次和第 1 天最大的区别是什么

第 1 天里，我们用的是 toy dataset，也就是人为构造的小数据。

第 2 天开始，你接触的是真正的图像数据集：

- 训练集：60000 张图片
- 测试集：10000 张图片
- 每张图大小：`28 x 28`
- 图片通道数：1
- 类别数：10

也就是说，模型第一次开始“看图片”了。

## 2. 为什么今天还用 MLP，不直接上 CNN

因为今天的重点不是卷积，而是先弄懂下面这条完整链路：

1. 数据文件从哪里来
2. 数据怎么读进内存
3. 图片怎么变成张量
4. DataLoader 怎么按 batch 提供数据
5. 模型怎么接收图片并输出 logits
6. 损失函数怎么比较预测和标签
7. 优化器怎么更新参数

如果现在就加 CNN，你会同时学太多新东西。

所以今天采用最稳的方案：

- 真实图片数据
- 简单 MLP 模型

## 3. 数据是怎么组织的

FashionMNIST 的每张图片本来是：

```text
1 x 28 x 28
```

这里的含义是：

- `1`：灰度图，只有 1 个通道
- `28 x 28`：图片高和宽

一个 batch 的图片会长这样：

```text
[batch_size, 1, 28, 28]
```

如果 `batch_size = 64`，那就是：

```text
[64, 1, 28, 28]
```

## 4. MLP 为什么要先 flatten

MLP 不能直接处理二维图片结构，它更擅长处理一维向量。

所以在进入全连接层之前，我们先把图片展平：

```text
1 x 28 x 28 -> 784
```

于是：

```text
[64, 1, 28, 28] -> [64, 784]
```

接着模型再输出 10 个类别的 logits：

```text
[64, 784] -> [64, 10]
```

## 5. logits 是什么

logits 可以理解成“每个类别的原始分数”。

比如一张图片经过模型后，输出可能像这样：

```text
[1.2, -0.4, 0.8, 2.7, ...]
```

这不一定是概率，但它表达了模型更倾向于哪些类别。

`CrossEntropyLoss` 会拿这些 logits 和真实标签比较，告诉模型这次错得有多严重。

## 6. 一次训练 batch 里到底发生了什么

每次训练一个 batch，核心顺序都是：

```python
logits = model(images)
loss = criterion(logits, labels)
optimizer.zero_grad()
loss.backward()
optimizer.step()
```

你可以把它翻译成人话：

1. 模型先看图片并做出预测
2. 计算这次预测错了多少
3. 清掉上一次遗留的梯度
4. 根据这次错误反向传播
5. 用梯度更新模型参数

## 7. train 和 test 有什么区别

训练阶段：

- 使用训练集
- 会执行 `backward()`
- 会执行 `optimizer.step()`
- 参数会改变

测试阶段：

- 使用测试集
- 只看表现，不更新参数
- 通常放在 `torch.no_grad()` 里

所以你可以理解成：

- `train` 是学习
- `test` 是考试

## 8. 今天你应该重点看哪些输出

你运行程序后，先看这些基础信息：

- `Image batch shape`
- `Label batch shape`
- `Logits shape`
- `Model parameters`

然后每个 epoch 看：

- `train_loss`
- `train_acc`
- `test_loss`
- `test_acc`

正常情况下：

- `loss` 应该逐渐下降
- `accuracy` 应该逐渐上升

## 9. 如果准确率不高，是不是代码错了

不一定。

因为今天的模型只是一个简单 MLP，不是 CNN。
它能跑通真实图像任务已经很好了，但准确率通常不会像后面 CNN 那么高。

今天的重点是：

- 数据读对了
- shape 对了
- 训练循环对了
- 能得到一版合理结果

这就已经完成第 2 天任务了。

## 10. 第 3 天会学什么

第 3 天最自然的升级是：

- 把 `MLP` 换成 `CNN`
- 不再先粗暴展平整张图片
- 让卷积层去提取图像局部特征

你会明显看到：

- CNN 更适合图像
- 准确率通常比简单 MLP 更高

也就是说，第 2 天是在打地基，第 3 天才是正式进入“图像深度学习模型设计”。
